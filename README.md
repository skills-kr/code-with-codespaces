# GitHub Codespaces로 코딩하기

_Codespaces를 사용하여 사전 구성된 개발 환경에서 프로젝트를 개발하세요!_

## 환영합니다

GitHub Codespaces는 클라우드에서 호스팅되는 개발 환경입니다.

- **대상**: 개발자, DevOps 엔지니어, 엔지니어링 매니저, 프로덕트 매니저
- **배울 내용**: Codespace를 생성하고, 구성하고, 사용하는 방법
- **만들 것**: 저장소에 맞춤화된 Codespace 구성
- **사전 요구사항**: [Visual Studio Code 문서](https://code.visualstudio.com/docs)에 익숙할 것
- **소요 시간**: 이 실습은 1시간 이내에 완료할 수 있습니다.

이 실습에서 다음을 수행합니다:

1. 첫 번째 Codespace를 만들고 코드 변경하기
1. Codespace 커스터마이징하기
1. Codespace 사용 연습하기

### 이 실습을 시작하는 방법

실습을 내 계정으로 복사한 뒤, Octocat(Mona)이 첫 번째 레슨을 준비할 때까지 **약 20초** 기다린 후 **페이지를 새로고침**하세요.

실습을 복사할 때 다음 설정을 권장합니다:

- 소유자(owner)는 개인 계정 또는 조직(organization)을 선택하세요.
- 비공개 저장소는 Actions 사용 시간이 소모되므로 공개 저장소를 만드는 것을 권장합니다.

20초 후에도 실습이 준비되지 않으면:

1. 새 저장소가 생성된 후 약 20초 기다린 다음 페이지를 새로고침하세요.
2. 저장소에 생성된 이슈의 단계별 지침을 따르세요.
3. 페이지가 자동으로 새로고침되지 않으면 [Actions](../../actions) 탭을 확인하세요.
